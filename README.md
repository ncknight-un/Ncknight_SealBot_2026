# Final Project – ROS2 Workspace

## Overview
This repository contains a ROS2 workspace for the Final Project, including the `Sealbot` package and supporting data, scripts, and documentation.

The goal of this project is to develop and manage a robotic system using ROS2, including data handling, package structure, and deployment workflows.
